# shoya.io

My portfolio and weblog using [Next.js](https://nextjs.org/)

## Getting Started

Compiles and hot-reloads for development

```
$ yarn dev
```

Compiles and minifies for production

```
$ yarn build
```

Run a web server with the production build

```
$ yarn start
```
